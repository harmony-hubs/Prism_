# Harmony Heart Beats
    
    ## Vision
    Spreading harmony across blockchains using a multi-chain musical NFT collection.
    
    ## Tech Stack
    - **Blockchain:** Sui
    - **Infrastructure:** Ika (formerly dWallet Network) for sub-second Parallel MPC and dWallets.
    - **Onboarding:** WaaP (Wallet-as-a-Protocol) for social/email login and gas-free transactions.
    - **Interface:** Integration with HarmonyDAW.studio.
    
    ## Core Concept
    Each NFT is a programmable dWallet ('Omni-Heart') that can natively control assets on Bitcoin, Ethereum, and other chains to send 'pulses' of harmony.
